# Charlie Unified Overview

## Layers

### 1. Core

- Located in `/core`
- Handles prompts, workflows, and configuration

### 2. CVRCC

- Located in `/cvrcc`
- Handles command routing, validation, coordination, and recovery

### 3. Android

- Located in `/android`
- Handles ADB flows, Termux automation, and device-side control

### 4. Nexus

- Located in `/nexus`
- Handles synchronization, state exchange, and messaging

### 5. Aegis

- Located in `/aegis`
- Handles integrity checks, logging, and security policy
